# Support

## Getting Help

### Bug Reports
If SupaHunt crashes, returns unexpected results, or fails on a specific target type, [open an issue](https://github.com/MikeRoss27/supahunt/issues/new?template=bug_report.md) with:
- Python version (`python3 --version`)
- The command you ran
- Error output / traceback
- Target framework (Next.js, Vite, React, SvelteKit, etc.)

### Feature Requests
Want a new attack module, payload, or detection technique? [Open a feature request](https://github.com/MikeRoss27/supahunt/issues/new?template=feature_request.md).

### Questions & Discussions
For general questions about Supabase security testing, usage tips, or attack techniques:
- Check existing [issues](https://github.com/MikeRoss27/supahunt/issues) first
- Open a new issue with the `question` label

### Security Vulnerabilities
If you found a security issue **in SupaHunt itself** (not in a target you scanned), see [SECURITY.md](SECURITY.md).

## Disclaimer

SupaHunt is provided as-is for authorized security testing. The maintainers do not provide support for using this tool against systems you do not have permission to test.
