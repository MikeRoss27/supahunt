# Code of Conduct

## Our Standards

This project exists to improve the security of Supabase applications. Contributors and users are expected to:

- Use this tool only with explicit authorization from system owners
- Report discovered vulnerabilities responsibly
- Not share exploit data, credentials, or PII found during testing
- Respect the legal frameworks of their jurisdiction
- Help improve detection and remediation, not enable attacks

## Unacceptable Behavior

- Using this tool against systems without authorization
- Sharing real target data (credentials, PII, database dumps) in issues or PRs
- Promoting illegal activity
- Harassment of any kind

## Enforcement

Violations may result in being banned from the project. Illegal use should be reported to appropriate authorities.

## Contact

Open an issue or use GitHub's private reporting feature.
